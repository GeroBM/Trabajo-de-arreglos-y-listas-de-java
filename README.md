# 🎯 Objetivo del Proyecto

> **Propósito Central:** Que el estudiante comprenda el concepto de **pila** y su **estructura lógica**, siendo capaz de aplicarlo en un simulador de gestión de pedidos (**Undo/Redo**) para una pizzería, implementado en **Java**.

---

### 🚀 Alcance y Evaluación
La actividad busca evaluar la capacidad del estudiante para:

* **Combinar estructuras:** Implementación de arreglos y listas ligadas.
* **Colaboración:** Trabajar coordinadamente en equipo.
* **Control de Versiones:** Aplicar buenas prácticas mediante **GitHub**.
